# Text Counter

A simple Python command-line tool to analyze a text and display basic metrics.

## Features

- Count total characters (with spaces)
- Count characters without spaces
- Count words
- Count lines
- Estimate number of sentences
- Show a full summary
- Display letter frequency (A–Z)

## Requirements

- Python 3.9+ recommended

## How to Run

1. Clone the repository and enter the project directory:
   ```bash
   git clone <YOUR_REPOSITORY_URL>
   cd <PROJECT_FOLDER>

2. Run the script:
    ```bash
    python word_counter.py
    ```

## Notes

Sentence counting is an approximation based on ., !, and ?.

Word counting uses basic whitespace splitting.
